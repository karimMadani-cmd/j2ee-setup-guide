# Projet Jakarta EE

## Description
Ce projet est un exemple d'application web développée avec **Jakarta EE**. Il permet de comprendre et de pratiquer les concepts fondamentaux des applications Java côté serveur, notamment :

- Les **Servlets** pour gérer les requêtes HTTP.
- Les **JSP (JavaServer Pages)** pour créer des pages web dynamiques.
- La gestion des **sessions** et des **cookies** pour suivre l'état des utilisateurs.

Le projet illustre également l'intégration avec un **serveur web** et l'utilisation d'un **IDE** pour le développement.

## Technologies utilisées
- **Eclipse IDE** : environnement de développement pour Java EE/Jakarta EE.
- **Apache Tomcat** : serveur d'application pour exécuter les Servlets et JSP.
- **Google Chrome** : navigateur pour tester et visualiser l'application web.

## Installation et exécution
1. Cloner ou télécharger le projet dans votre ordinateur.
2. Ouvrir le projet avec **Eclipse IDE**.
3. Configurer le serveur **Tomcat** dans Eclipse.
4. Déployer le projet sur Tomcat.
5. Ouvrir **Google Chrome** et accéder à l'URL : `http://localhost:8080/nom_du_projet`

## Objectifs du projet
- Découvrir le cycle de vie d’une Servlet.
- Manipuler les données envoyées via un formulaire web.
- Générer des pages web dynamiques avec JSP.
- Comprendre l’architecture client-serveur pour les applications web Java.
